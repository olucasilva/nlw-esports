# eSpots Next Level Week - intensivão de uma semana pela RocketSeat

Neste projeto foram desenvolvidas as aplicações do servidor, uma página web no modelo SPA e um app mobile. Usando conceitos de componentes e propiedades do ReactJS para web, com estilização usando tailwind e ReactNative para mobile. No server side foi empregado NodeJS com Express, banco de dados SQLite e Prisma como ORM.

Foram usadas bibliotecas do RadixUI na criação de um modal acessível na web.

Foi utilizado Typescript para desenvolvimento.
